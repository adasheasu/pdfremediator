üîß Troubleshooting

‚ùå "This site can't be reached" or "Connection refused"

Solutions:

  • Make sure the tool is still running on your computer
  • Check that both computers are on the same WiFi network
  • Double-check the IP address is correct
  • Try turning off VPN if you have one
