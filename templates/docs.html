FRAME (Faculty Resources for Accessible Media Essentials)

FRAME is a Canvas center providing videos, step-by-step instructions, and accessibility resources specifically for faculty.

  • Access: FRAME Resources
  • Content includes: Video tutorials, written guides, accessibility best practices
  • Workshops: Regular workshops available - check the site for schedule

LinkedIn Learning Courses

Free for ASU faculty and staff! Continue your accessibility education with these comprehensive courses:

  • "Digital Accessibility for the Modern Workplace" (3 hours) - Covers accessible meetings, presentations, and communications
  • "Creating Accessible PDFs" (5.5 hours) - Learn Word, PowerPoint, and InDesign methods
  • "Microsoft 365: Making Documents Accessible" - Video series on making Office documents accessible

  • Compliance Level: WCAG 2.1 Level AA required for all ASU digital content

üîß Additional Tools & Services

PDF Remediation Tools

  • Equidox - Free cloud-based PDF remediation tool for ASU faculty/staff
    • Better for larger size and volume of PDFs
    • Request access via ServiceNow
    • ASU Equidox Guide
  • Adobe Acrobat Pro - Good for smaller PDFs and fewer number of PDFs
    • Manual remediation with powerful tools
    • Adobe Guide

Testing Tools

  • WAVE Browser Extension - Provides initial assessment (catches 20-30% of issues)
    • Get WAVE Extension
  • Siteimprove - Continuous monitoring
    • ASU units can request free scanning
    • ASU Siteimprove Info
  • ASU Web Accessibility Audit Tool - Step-by-step guided evaluation
    • Access Audit Tool

Screen Readers

  • JAWS (Windows) - Free for ASU faculty/staff via Freedom Scientific portal (40.5% of users)
  • NVDA (Windows) - Free open-source screen reader (37.7% of users) - Download
  • VoiceOver (Mac/iOS) - Pre-installed on Apple devices (70.6% of mobile users)

Canvas Tools

  • Ally - Automated accessibility checker for Canvas content
  • MediaPlus - ASU tool for adding captions and transcripts to videos
  • Equatio - Digital equation accessibility tool for STEM content
